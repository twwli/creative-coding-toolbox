# Workshop Toolbox

Ce dépôt GitHub accompagne le Workshop Creative Coding donné à LISAA Strasbourg en juin et juillet 2026.

📚 **Cours :** [lisaa.olivierguillard.dev/g4](https://lisaa.olivierguillard.dev/g4/index.html)

---

## Installation

### VS Code

Installez [VS Code](https://code.visualstudio.com), ainsi que les extensions suivantes :

- **Live Server** (par Ritwick Dey)
- **Claude Code for VS Code** (par Anthropic)
- **Codex** (par OpenAI)

### GitHub

- [Créez un compte GitHub](https://github.com)
- Installez [GitHub Desktop](https://docs.github.com/en/desktop/installing-and-authenticating-to-github-desktop/installing-github-desktop)

## Boilerplate HTML

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Mon Projet</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <!-- Ici le contenu de votre projet -->
    <script src="scripts.js"></script>
  </body>
</html>
```

**CSS Reset :** [github.com/twwli/CSS-reset](https://github.com/twwli/CSS-reset)
